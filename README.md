# PBHLT_7101
Github repository for the PBHLT 7101 class for Darshan Shimpi.
Currently contains files for project 1 including the data files, do files used throughout the project, write-up, and other files that were used in the project overall. 

## Project 1
To run the code for project 1, simply download the ansur dataset provided as well as the do file. Some packages may need to be installed such as violinplot or others. 
The project_1_code.do file will create the v0-v4 and the final dataset when run.
The project_1_figures.do file will generate the visuals and tables for the write-up.
